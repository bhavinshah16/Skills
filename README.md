# Claude Skills

A collection of custom Claude skills for enhanced productivity.

## Available Skills

### [Investment Analyzer](./investment-analyzer)

Comprehensive investment analysis for evaluating real estate and business deals.

**Features:**
- Analyze NNN (Triple Net) commercial properties
- Calculate key metrics (Cap Rate, NOI, Cash-on-Cash Return, IRR, NPV)
- Score deals against industry benchmarks
- Get BUY / NEGOTIATE / PASS recommendations

**How to Use:**
See the [Investment Analyzer README](./investment-analyzer/README.md) for detailed instructions

---

## Installation

To use these skills:

1. Visit the skill folder you want to install
2. Follow the instructions in that skill's README.md
3. Copy the skill to your Claude.ai or Claude app environment as needed

## Adding More Skills

To add a new skill to this collection:

1. Create a new folder in the root with your skill name
2. Add your SKILL.md and supporting files
3. Create a README.md in that folder with installation/usage instructions
4. Update this main README.md with a reference to the new skill
